Capture Workflows

Three capture modes all accessible from the toolbar

1. Window Selection (3 methods)
📋 Dropdown
List of all open windows, auto-refreshed. Click to select. Filter box at top.
🎯 Crosshair
Click and drag a transparent overlay to any window. Window title captured on release.
⌨ Manual
Type a window title or process name. Partial match supported. Regex toggle.
2. Template Image Capture
📸 Screen Region Capture
1. Click "Capture Template" button
2. Screen freezes with a semi-transparent overlay
3. Drag to select the region
4. Preview + name the template → saved to project/images/
📂 Import from File
1. Click "Import" button
2. File dialog → select PNG/JPG/BMP
3. Preview + optional crop
4. Name → saved to project/images/
3. Coordinate Picker
🖱 Click-to-Pick Coordinates
1. In an action property (e.g., "Click Coordinate"), there's a 📍 Pick button
2. Clicking it minimizes the tool window and shows a crosshair cursor
3. Press Space anywhere on screen — coordinates auto-filled
4. Or press Esc to cancel
5. Shows a small floating preview: X: 845, Y: 420 | RGB: 234,187,45
+ Also works while a script is running — for quick coordinate calibration

Template Management

Template Library Panel
• Thumbnail grid of all project templates
• Shows: name, dimensions, file size, last used date
• Right-click: rename, delete, duplicate, test-match
• Drag a template directly into the script tree to create a trigger
• Test-match: runs template matching on current screen, shows confidence score heatmap
