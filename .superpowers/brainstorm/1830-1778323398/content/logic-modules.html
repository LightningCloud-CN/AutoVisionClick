How Beginners Learn It

1. Template Gallery

Built-in starter templates (Auto Click, Heal Check, etc.) that new users can load and inspect. Each template has explanatory comments.

2. Wizard Mode

Answer 3 questions ("Which image?", "What to do?", "How often?") and the wizard generates the script. Can be edited later.

3. Tooltips & Validation

Every module has a hover tooltip. Red highlights show incomplete modules. Yellow warnings for common mistakes.
